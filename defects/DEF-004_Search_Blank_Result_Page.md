# Invalid search returns blank page

Environment
Browser: Chrome
Website: https://automationexercise.com

Steps to Reproduce
1. Navigate to https://automationexercise.com
2. Click on the ""Products"" Button at the top of the page
3. Scroll down to the ""ALL PRODUCTS"" section
4. Search term: ""XYZNonExistingProduct""

Expected Result
<br>
System displays a clear message indicating that no products match the search query.

Actual Result
<br>
System displays a blank webpage.

Severity: Low
Priority: Low

Impact
<br>
Not indicating the reason for a blank webpage can result in delays for purchases
